


안녕하세요 저는 이번에 발표를 하게된 이예지라고 합니다.
지금부터 최종 발표를 시작하곘습니다.

제가 1차에서 제작한 사이트는 
동그라미 인형공방이라고 하는 자수 제작업체의 사이트입니다.
동그라미는 현재 홈페이지가 없고 
트위터, 블로그를 통해 고객과 소통하고 있는 것이 특징입니다.
제가 개인적으로 아는 지인의 의뢰를 받아 사이트 제작을 결심하게 되었습니다.

처음 동그라미 인형 사이트를 분석해보니
트위터와 블로그만 이용하다보니 정보가 여기저기 흩어져 있다는 단점을 발견하였습니다.
제일 중요한 가격 정보에 접근하기도 힘든 상황이었습니다.

그래서 저는 사이트의 장단점을 찾고 
합목적성, 일관성, 사용성, 심미성 측면에서 점수를 매겨봄으로써 
사이트를 심층적으로 분석 해보았습니다.

또한 참고가 될만한 다른 사이트들을 함께 분석하여 보았는데요.
ourplanet 이라고 하는 netflix Tv 시리즈의 페이지나 
디자인이 예쁜 홈페이지들을 모아보는 사이트,
그리고 에버노트 등을 참고하였습니다.

이러한 분석을 통해 막연하게만 생각하던 현재 사이트의 상황을 더 잘 알게 되었고
리뉴얼을 통해 떨어지는 부분들을 개선하기로 하였습니다.

그래서 이런 분석들을 통해 제가 잡은 목표는
정보를 편하게 찾아갈 수 있는 사이트. 
채도를 낮춘 배색을 가진 사이트
동그라미와 라인을 디자인 컨셉으로 하는 사이트를 만드는 것입니다.

목표를 잡은 뒤 저는 스토리 보드를 작성하여 사이트를 기획하였고요.
스타일 가이드를 만들어 디자인을 하였습니다.
자세한 디자인 관련해서는 실제 동작 화면에서 자세히 설명해 드리도록 하겠습니다. 

현재 제작 완료된 페이지는 
메인페이지, 소개페이지, 제작공정 페이지, 샘플페이지, 로그인 및 가입하기 페이지입니다.

개발 관련해서 소개하자면 
모든 페이지에서 웹표준을 준수하기 위해 노력하였고 
LESS 를 사용하여 CSS 의 가독성을 높이고자 하였습니다. 
또한 OG TAG를 작성하여 링크를 공유할 때 정보를 불러올 수 있게 하였습니다. 
페이지에 들어간 아이콘이나 로고같은 작은 이미지들은 
다음과 같이 스프라이트 이미지를 만들어 모아놓음으로서 페이지 로딩 시간을 줄여주었습니다.




여기까지 2분 30초




그럼 실제 구현 내용을 설명하도록 하겠습니다. 

먼저 사이트 전체적으로 폰트는 귀여운 느낌을 주는 빙그레따옴체 사용하였고
배색은 채도를 낮춘 붉은색 계통으로 색을 잡았습니다. 

페이지의 로고는 직접 제작하였고 바늘땀을 연상시키도록 라인을 이용하여 디자인 하였습니다.
로고 위에 마우스를 올리면 라인이 채워지는 애니메이션 효과를 주었습니다. 

헤더 부분은 마우스를 호버하면 나오게 되어있고요.
메뉴 자체를 누르거나 안쪽 서브 메뉴를 누르면 페이지가 이동하게 되는데요. 
현재 누른 페이지가 어디인지 알 수 있도록 누르면 가게 되는 페이지를 분홍색으로 하이라이트 컬러를 주었습니다.

메인 위쪽엔 배너가 있고 슬라이드 라이브러리를 사용해 구현하였습니다.
슬라이드 위에 링크가 있는데요. 
사이트 전체적으로 버튼은 애프터 선택자를 이용하여 마우스를 오버하면 채워지는 형식으로 구현하였습니다.

아래 간이 메뉴의 경우 각 페이지로 바로 이동할 수 있는 링크인데요. 
링크라는 것을 사용자에게 알려주기 위해 마우스를 링크 영역에 가져갔을 때 색상을 반전해주고
버튼 위에 마우스를 오버하면 한 번 더 색상을 반전해주어 마우스 위치를 확실하게 알 수 있게 해주었습니다. 

전체적으로 내용들은 스크롤을 하면 스르륵 나타나게 되어있고요. 
다시 위로 올렸을 때는 들어가지 않습니다. 

각 사진들은 마우스오버를 하면 뒤의 이미지가 조금 커지면서 버튼이 나타나도록 구성하였습니다. 

이렇게 페이지 끝으로 오면 헤더로 돌아가기 위해 스크롤을 한참 올려야 하는데요. 
왼쪽 상단에 해버거버튼이 따라오게 만들어서 누르면 이렇게 헤더가 내려오도록 하여 
쉽게 네비게이션 메뉴에 접근할 수 있도록 하였습니다. 

다음은 소개 페이지입니다.
소개 페이지에서는 스크롤을 내렸을 떄 밤이던 건물이 밝아지거나 
생각하는 인형의 전구에 불이 들어오는 등 사진 편집을 통해 여러 효과들을 주었습니다. 

맨 아래에는 map api 를 이용하여 회사의 위치를 알려주었습니다. 

다음은 제작공정 페이지입니다.
제작공정 페이지에서는 스크롤을 내리면 뒤의 배경화면과 내용이 바뀌도록 구성하였는데요.
좌측 서브메뉴를 눈금자처럼 만들었고 자수를 실제 제작하는 내용을 비디오로 보여주기도 하였습니다.

다음은 인형 사진 페이지인데요.
2 컬럼으로 되어있고 스크롤을 내리면 더보기 형태로 사진이 붙도록 되어 있습니다. 
사진이 붙는 내용은 직접 javascript 로 구현하였는데요. 
새로 붙는 사진의 높이와 현재 컨테이너의 높이를 비교해서 더 짧은 쪽에 사진이 붙도록 하였습니다.
사진은 클릭하면 크게 볼 수 있습니다. 
오른쪽 셀렉트박스의 경우 사이트의 디자인에 맞추어 디자인을 커스텀 해주었습니다. 

마지막으로 로그인 페이지의 경우 폼을 사이트 디자인에 맞추어 꾸며주었고 
폼 안에 아무것도 입력하지 않고 로그인 버튼을 누르면 
유효성 체크를 해서 틀린 폼이 흔들리면서 에러 메시지가 뜨게 됩니다. 
가입 페이지도 똑같은 형식으로 구현되어 있습니다.
